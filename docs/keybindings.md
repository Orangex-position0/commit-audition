# Vim Mode 快捷键

## 通用导航

| 按键 | 功能 |
|------|------|
| `h` / `←` | 上一步 |
| `l` / `→` | 下一步 |
| `Ctrl+s` | 跳过当前步骤，进入下一步 |

## 列表选择（类型 / 正文编辑方式）

| 按键 | 功能 |
|------|------|
| `j` / `↓` | 向下移动光标 |
| `k` / `↑` | 向上移动光标 |
| `Enter` | 确认选择并进入下一步 |
| `/` | 进入搜索模式 |
| `Esc` | 退出搜索 → 退出程序（级联） |
| `q` | 退出程序 |

## 文本输入（标题 / Issue 编号）

| 按键 | 功能 |
|------|------|
| 任意字符 | 输入字符 |
| `Backspace` | 删除末尾字符 |
| `Enter` | 确认并进入下一步 |
| `Esc` | 清空输入 → 退出程序（级联） |

## 搜索模式（列表中按 `/` 激活）

| 按键 | 功能 |
|------|------|
| 任意字符 | 追加搜索关键词 |
| `Backspace` | 删除末尾搜索字符 |
| `Enter` | 确认当前选中项并退出搜索 |
| `Esc` | 退出搜索模式 |

## 预览确认

| 按键 | 功能 |
|------|------|
| `y` / `Enter` | 确认输出 commit message |
| `n` | 返回第一步重新编辑 |
| `q` / `Esc` | 取消并退出程序 |

## Esc 级联行为

按下 `Esc` 时，按以下顺序依次响应：

1. **列表步骤**：先退出搜索 → 再退出程序
2. **输入步骤**：先清空输入 → 再退出程序
3. **预览步骤**：直接退出程序
