# Vim Mode 快捷键

Vim 模式采用 lazygit 风格的**查看/编辑模式**分离设计：

- **查看模式**：vim 按键（h/j/k/l/q/y/n）作为快捷键生效
- **编辑模式**：所有按键作为普通文本输入，按 `Esc` 返回查看模式

## 通用导航（查看模式）

| 按键 | 功能 |
|------|------|
| `h` / `←` | 上一步 |
| `l` / `→` | 下一步 |
| `1`-`5` | 直接跳转到步骤 1-5 |
| `Ctrl+s` | 推进到下一步（重置编辑状态） |
| `q` | 退出程序 |

## 列表选择（类型 / 正文编辑方式）

| 按键 | 功能 |
|------|------|
| `j` / `↓` | 向下移动光标（循环） |
| `k` / `↑` | 向上移动光标（循环） |
| `Enter` | 确认选择并进入下一步 |
| `/` | 进入搜索模式 |
| `Esc` | 退出搜索 → 退出程序（级联） |
| `q` | 退出程序 |

## 文本输入 — 查看模式（标题 / Issue 编号）

| 按键 | 功能 |
|------|------|
| `Enter` | 进入编辑模式 |
| `h` / `l` | 切换步骤 |
| `1`-`5` | 跳转步骤 |
| `q` | 退出程序 |
| `Esc` | 清空输入（非空时） → 退出程序（为空时） |

## 文本输入 — 编辑模式（标题 / Issue 编号）

| 按键 | 功能 |
|------|------|
| 任意字符 | 输入字符（包括 h/j/k/l/q/y/n） |
| `Backspace` | 删除末尾字符 |
| `Enter` | 确认并推进到下一步 |
| `Esc` | 退出编辑模式（停留当前步骤，保留内容） |

编辑模式激活时，输入框边框变为**青色**。

## 搜索模式（列表中按 `/` 激活）

| 按键 | 功能 |
|------|------|
| 任意字符 | 追加搜索关键词 |
| `Backspace` | 删除末尾搜索字符 |
| `Enter` | 确认当前选中项并退出搜索 |
| `Esc` | 退出搜索模式 |

## 预览确认

| 按键 | 功能 |
|------|------|
| `y` / `Enter` | 确认输出 commit message |
| `n` | 返回第一步重新编辑 |
| `1`-`5` | 跳转步骤 |
| `q` / `Esc` | 取消并退出程序 |

## 步骤 3 双区域布局

步骤 3（正文编辑方式）显示两个区域：

- **上方区域**：编辑方式选择列表（可用 j/k 导航）
- **下方区域**：正文内容只读预览（未编辑时显示"尚未编辑正文"）

## Esc 级联行为

按下 `Esc` 时，按以下顺序依次响应：

1. **编辑模式**（任意步骤）：退出编辑模式，保留内容
2. **搜索模式**（列表步骤）：退出搜索 → 再退出程序
3. **输入步骤**（查看模式）：清空输入（非空时） → 再退出程序（为空时）
4. **预览步骤**：直接退出程序
